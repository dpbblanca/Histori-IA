Privacidad y RGPD

Histori-IA no solicita nombre, correo electrónico, contraseñas, imágenes ni datos identificativos.

El recurso está pensado para consulta directa mediante URL pública. No se recopilan, almacenan ni procesan datos personales de ningún tipo.

Si en el futuro se integran cuestionarios de evaluación, estos deberán aplicarse de forma completamente anónima o mediante códigos no identificativos, sin vincular resultados a datos personales.

El audio generado mediante Web Speech API se procesa localmente en tu dispositivo. No se registran ni envían transcripciones a servidores externos.

Ayuda y Uso

Cómo usar Histori-IA

  • Busca: Escribe en el buscador para localizar contenidos por palabras clave.
  • Selecciona: Elige un contenido del desplegable de "Contenidos Disponibles".
  • Lee: Consulta la descripción, utilidad didáctica y mapa conceptual.
  • Escucha: Haz clic en "Escuchar" para oír una explicación en voz alta.
  • Descarga: Usa "Descargar PDF" para guardar el contenido.
  • Contraste: El botón "🌓 Contraste" cambia entre tema claro y oscuro.

Recomendaciones Pedagógicas

  • Usar como recurso complementario en clase o estudio independiente.
  • Lectura de mapas conceptuales para visualizar relaciones causales y jerarquías.
  • Escucha del audio para reforzar conceptos de forma multisensorial.
  • Descarga de contenidos para incluir en trabajos, esquemas o presentaciones.
